Current order list

The order list starts empty. Click "Generate list" to auto-fill it with items that are low or out of stock — each one comes with a recommended box count based on how it's been selling. Use the search above to add any other items manually. Items are grouped by category.

Checkboxes let you hold things back without losing them. Uncheck an individual item (or a whole category using the checkbox in its header) to keep it on the list but leave it out of this week's order. When you click "Mark as ordered," only the checked items go to Order arrivals — anything unchecked stays right here for next time. The ✕ button removes an item from the list entirely.

Use the category dropdown in the toolbar when you need to focus on one supplier (like ice cream from a separate vendor).

Box sizes

Set how many individual units come in one box for each item (e.g. 12 if a case has 12 bags). This is saved permanently until you change it. Box sizes are used to suggest accurate box counts in the order list and Suggested reorder column.

Orders you've placed appear here as "Pending arrival" (yellow border). When a delivery shows up, click "Confirm arrival," check that what came matches what you ordered, edit any quantities that differ, then click "Mark as received." Your stock updates automatically.

The Clover import file isn't downloaded automatically — when you're ready to sync to Clover, click the yellow "↓ Download Clover file" button on the most recent received order (marked with a ★ "Latest · most accurate" badge). That file has your latest stock counts and is the right one to upload to Clover. Older received orders keep download buttons too in case you ever need to re-grab one, but they're labeled "older" since their counts are outdated.

Received orders stay here as a permanent history (green border) until you click "Remove" on them.

Slow movers

Items that aren't selling. The site looks at sales across the most recent uploads and surfaces anything still on the shelf but barely moving. Use this to spot dead stock before reordering it, items to discount, or things to discontinue. Items need at least 2 uploads of history to appear here.

Sync check

Catches potential data-entry errors and drift between Clover and this site. Items get flagged if their stock changed in ways that don't match what was sold + what was restocked, if sales spiked far above normal, or if more was sold than was actually on the shelf. Review each flagged item to see if Clover got an incorrect adjustment.

How this works

This site is shared. Everyone with the link sees and edits the same live data — no separate copies, no spreadsheet uploads needed for viewers.

The weekly routine:

  1. In Clover, export your inventory (the multi-tab Excel file).
  2. Click "Upload export." It updates stock for everyone, picks up any new items you added in Clover, and compares to last week.
  3. Check the weekly summary, then the Order list tab to see what to buy.

Box sizes: visit the "Box sizes" tab to set how many individual units come in one box for each item (e.g. 12 if a case holds 12 bags). This saves permanently and powers all the box-based suggestions. You can also set the box size in the "+ Add" restock popup.

Smarter reorder suggestions: the suggested box count adapts to how each item is selling. Items selling fast get bigger reorders; slow movers get smaller ones. Always rounded up to whole boxes.

Logging deliveries: when stock arrives, the cleanest path is the Order list → Order arrivals workflow described below. For a one-off "I just got more of this one item" situation, click "＋ Add" on that row to enter boxes, units per box, and loose units.

Quick Clover download: the "↓ Download Clover file" button in the header at any point downloads a Clover-ready import spreadsheet reflecting the current state of your stock. Useful any time Clover and the website have drifted out of sync.

Order list & arrivals: items below their alert threshold show up automatically in the Order list tab. Add others manually with the search. When you actually place the order, hit "Mark as ordered" — it goes to the Order arrivals tab as a pending arrival and the order list clears for next time.

Backups: data lives in cloud storage and is reliable, but "Download backup" gives you a safety file you can keep in Google Drive just in case.

Store hours (used by the upload warning)

These hours control when the "data may be outdated" warning appears at the top of the page. Set the open and close time for each day. Check the box to mark a day as closed.
